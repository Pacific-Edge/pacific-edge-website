Pacific Edge AI — Integration logos
===================================

Drop the official logo file for each platform in THIS folder, named exactly as
listed below (all lowercase). SVG is best. PNG with a TRANSPARENT background
also works (just keep the same name, e.g. dentrix.png instead of dentrix.svg).

How it works: the homepage integrations wall is already built to use these.
The moment a file is here, its tile shows the real logo and the placeholder
initials hide automatically. Send Leone's developer the files (or just add them
here) and they'll be wired in, with any that render poorly flagged for a better
version.

WHERE TO GET THE OFFICIAL FILES
  Most companies publish them on a "Brand assets", "Press kit", "Media", or
  "Newsroom" page (e.g. search "<company> brand assets"). Since you have
  permission, the cleanest option is to ask your contact at each company to
  send the official SVG — highest quality, no guessing on the right version.

QUALITY TIPS
  - SVG = perfect at any size. Prefer it whenever offered.
  - If PNG: use one with a transparent background, at least ~200px tall.
  - Avoid screenshots or logos with a white/colored box around them.
